# Gravii Landing Site

Marketing site for Gravii (gravii.app). Built with Next.js, Tailwind CSS.
Repo: Pandosaure/gravii-site

## Structure
- src/app/page.tsx — main landing page (hero, demo, product screenshots, pricing)
- src/app/api/analyze/route.ts — AI demo endpoint (Anthropic Claude)
- public/images/ — all assets (icons, screenshots)

## Rules
- No em-dashes — use hyphens
- Always complete files, never partial edits
- Positioning: "Product memory" not "product intelligence"
- Don't say "zero input" — say "effortless"
- Keep "For B2B product teams" badge
